# Training Plan App

A web application for creating and managing training plans.

## Features

- Create and manage training plans
- Track progress and achievements
- User authentication and profile management
- Responsive design for all devices

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/training-plan-app.git
```

2. Install dependencies
```bash
cd training-plan-app
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Vite
- React Router
- React Query
